Safety database

This is an online safety database for reporting adverse drug reactions (ADR).
You can either submit a new ADR or view the ADRs that you have submitted.
The admins of the system can view all the ADRs.

If you have received information about a new ADR that has not been previously reported to the authorities or to the Marketing Authorisation Holder (the pharmaceutical company), please report the ADR using the ADR Form.

If you want to see information about the ADRs reported to the system, you can go to the ADR Information page.
